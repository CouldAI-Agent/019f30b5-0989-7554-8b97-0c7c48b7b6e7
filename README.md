# මාධ්‍ය ආචාර ධර්ම සහ මාර්ගෝපදේශ යෙදුම (Media Ethics App)

මෙම යෙදුම මගින් මාධ්‍ය සඳහා ආචාර ධර්ම සහ මාර්ගෝපදේශවල වැදගත්කම පැහැදිලි කෙරේ. පුවත්පත්, රූපවාහිනිය සහ නව මාධ්‍ය යන ප්‍රධාන අංශ තුන යටතේ මාධ්‍ය ආචාර ධර්ම ක්‍රියාත්මක වන ආකාරය පිළිබඳව මෙහි අන්තර්ගත වේ.

## අන්තර්ගතය
- **පුවත්පත් ආචාර ධර්ම**: සත්‍යතාව, අපක්ෂපාතීත්වය, මූලාශ්‍ර ආරක්ෂා කිරීම සහ පුද්ගලිකත්වයට ගරු කිරීම.
- **රූපවාහිනි ආචාර ධර්ම**: දෘශ්‍ය සාක්ෂි විකෘති නොකිරීම, ප්‍රචණ්ඩත්වය අවම කිරීම සහ සමබර පුවත් වාර්තාකරණය.
- **නව මාධ්‍ය ආචාර ධර්ම (ඩිජිටල්/සමාජ මාධ්‍ය)**: සාවද්‍ය තොරතුරු (Fake News) පැතිරවීම වැළැක්වීම, සයිබර් හිරිහැර නොකිරීම සහ කර්තෘ අයිතිය සුරැකීම.
- **මාර්ගෝපදේශවල වැදගත්කම**: මහජන විශ්වාසය තහවුරු කිරීම, සමාජ ගැටුම් අවම කිරීම සහ ප්‍රජාතන්ත්‍රවාදය සුරැකීම.
